Introduction

The Attractiveness Level Detection project aims to develop a machine learning model capable of estimating the attractiveness level of a person from a facial image. This project encompasses several key phases, including data collection and preprocessing, facial landmark extraction, model selection, training, and evaluation. The project's potential applications span across human-computer interaction, psychological research, and marketing and advertising sectors.

Datasets

Two datasets were utilized in this project:

  • SCUT-FBP5500 Database: This dataset contains 5500 images labeled with beauty scores ranging from 1 to 5. It includes images of 2000 Asian females, 2000 Asian males, 750 Caucasian females, and 750 Caucasian males. The dataset's extensive labeling and variety make it a suitable choice for this project.
  • MeBeauty Database: This dataset comprises 2550 images of individuals from diverse racial backgrounds, rated by approximately 300 individuals. The beauty scores range from 1 to 10, and the dataset's diversity enhances the robustness of the model by providing a wide range of attractiveness ratings.

Methods

Several machine learning techniques were employed for both regression and classification tasks:

  • Classification: K-Nearest Neighbors (KNN), Support Vector Machine (SVM), Random Forest (RF), and Decision Tree (DT).
  • Regression: K-Nearest Neighbors (KNN), Support Vector Regression (SVR), Random Forest Regression (RFR), Decision Tree Regression (DTR), and Linear Regression (LR).
  • Convolutional Neural Network (CNN) for the MeBeauty dataset.

Google MediaPipe was extensively used for facial landmark detection and feature extraction. MediaPipe is a cross-platform framework for building custom machine learning pipelines. It was crucial in aligning facial features during the data preprocessing phase.
